# Front-end Playground

Este repositório contém exemplos de layouts e interfaces que foram implementados utilizando diversas tecnologias e frameworks.

## Tecnologias

As seguintes tecnologias foram utilizadas nos projetos:

- HTML
- CSS
- Sass
- JavaScript
- WordPress
- Prismic CMS
- React
- Next.js
